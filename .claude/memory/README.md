# Memory — Heydo's Decision Log

> The accumulated decisions and learnings of the Heydo civilization. When the Supreme Architect
> or a sovereign makes a meaningful call, it is recorded here so we never re-litigate it and so
> future work inherits the context.

## How to use
- One file per significant decision, numbered: `NNNN-short-slug.md`.
- Record the **decision, the why, who made it, and what it affects.**
- Supersede (don't silently overwrite) old decisions; link the new one and mark the old as superseded.

## Entry template
```markdown
# NNNN — <Decision title>
- Date: <YYYY-MM-DD>
- Phase: <0-9>
- Decider: <Supreme Architect / Sovereign>
- Status: accepted | superseded by NNNN

## Decision
<what we decided>

## Why
<the rationale, tied to the vision / a USP / a moat>

## Affects
<which sovereigns/geniuses/files/phases>
```
